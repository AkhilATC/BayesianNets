# 🧠 4. Bayesian Policy Agent (Core Brain)

### Simple Online Bayesian Router (No pgmpy)

```python
import numpy as np

AGENTS = ["urs_gen", "urs_validate", "knowledge", "fallback"]

# success counts
success = np.ones((4,4))  # agents x (domain,intent)
failure = np.ones((4,4))

def bayesian_policy(domain, intent):
    idx = domain*2 + intent
    probs = success[:, idx] / (success[:, idx] + failure[:, idx])
    agent_id = np.argmax(probs)
    return AGENTS[agent_id], probs

def update_policy(domain, intent, agent_id, reward):
    idx = domain*2 + intent
    if reward:
        success[agent_id, idx] += 1
    else:
        failure[agent_id, idx] += 1
```

👉 This is a **Dynamic Bayesian Policy**.

# 🔗 5. LangGraph Policy Node

```python
from langgraph.graph import StateGraph

def policy_node(state: AgentState):
    domain = state["domain"]
    intent = state["intent"]

    agent, probs = bayesian_policy(domain, intent)
    state["chosen_agent"] = agent
    state["history"].append(f"Policy chose {agent} with {probs}")
    return state
```

# 🧩 8. Build LangGraph

```python
graph = StateGraph(AgentState)

graph.add_node("policy", policy_node)
graph.add_node("urs_gen", urs_gen_agent)
graph.add_node("urs_validate", urs_validate_agent)
graph.add_node("knowledge", knowledge_agent)
graph.add_node("fallback", fallback_agent)

graph.set_entry_point("policy")

graph.add_conditional_edges(
    "policy",
    router,
    {
        "urs_gen": "urs_gen",
        "urs_validate": "urs_validate",
        "knowledge": "knowledge",
        "fallback": "fallback"
    }
)

app = graph.compile()
```

# 🔁 9. Feedback Loop (Dynamic Bayesian Learning)

After agent execution, collect reward:

```python
def feedback(domain, intent, agent_name, reward):
    agent_id = AGENTS.index(agent_name)
    update_policy(domain, intent, agent_id, reward)
```
