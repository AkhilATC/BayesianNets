# 🧠 1. First: What is a Bayesian Network (BN)?

A **Bayesian Network** is:

👉 A graph that represents **cause → effect with probabilities**

Each node = random variable
Each edge = dependency
Each node has a **CPD (Conditional Probability Distribution)**

---

### Simple Example

```text
Rain → Traffic
```

[
P(Traffic \mid Rain)
]

If it rains → traffic likely.

---

# 🕒 2. What Makes It *Dynamic*?

A **Dynamic Bayesian Network** adds **time**.

Instead of one snapshot, we have:

```text
Time t → Time t+1 → Time t+2 ...
```

---

### Example: Weather Over Time

```text
Rain_t → Rain_{t+1}
```

[
P(Rain_{t+1} \mid Rain_t)
]

If it rains today, tomorrow also likely rain.

---

# 🧩 3. Core Idea: Temporal Probabilistic Causality

DBN assumes:

> The present depends on the past (Markov assumption)

[
P(X_{t+1} \mid X_t, X_{t-1}, ...) \approx P(X_{t+1} \mid X_t)
]

👉 Only last state matters (simplifies math).

---

# 🌳 4. DBN Structure (Two-Slice Template)

You define:

### Slice at time t

### Slice at time t+1

Edges only go forward in time.

```text
X_t  →  X_{t+1}
Y_t  →  Y_{t+1}
X_t  →  Y_{t+1}
```

# 🧠 6. Key Intuition: Learning Over Time

### At time t:

[
P(R_t \mid D_t, I_t)
]

You choose an agent.

---

### After execution:

[
S_t = success / failure
]

---

### Update belief:

[
P(R_{t+1}) = P(R_t \mid S_t)
]

👉 If agent fails → reduce probability next time
👉 If agent succeeds → increase probability

# 🧮 8. Mathematical Core (Conceptual)

### Transition model:

[
P(R_{t+1} \mid R_t, S_t)
]

### Observation model:

[
P(S_t \mid R_t)
]

---

This is **same math as Hidden Markov Model (HMM)** but with multiple variables.

---

# 🧠 9. Belief Updating (Bayes Rule Over Time)

Core formula:

[
P(X_t \mid evidence) = \frac{P(evidence \mid X_t) P(X_t)}{P(evidence)}
]

In DBN, we repeat this every time step.

This is called:

👉 **Filtering**
👉 **Forward inference**
